自定义分页（非框架自带分页）

添加方式
添加class="page"，并添加列表容器list="#movies"等配置属性
< class="page" list="#movies" url="/list.json" items="dd" number="1" size="8" firstpage="false"></div>
属性 示例 说明 备注
list list="#movies" 数据容器（class或id） 必选
url url="/list.json" 获取分页的服务器地址 必选
items items="dd" 替换的标签（class或id或选择器） 可选，默认替换list下全部
number number="1" 起始页数 可选，默认值1
size size="8" 每页条数 可选，默认值10
firstpage firstpage="true" 是否生成第一页数据（如果为空则需给page添加属性recordCount pageNumber） 可选，默认值true
